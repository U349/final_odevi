# MotoWorld - Motosiklet Dünyası Bilgi Sistemi

MotoWorld, motosiklet kültürünü, farklı motosiklet tarzlarını, motor bloklarının çalışma prensiplerini ve dünya çapındaki ikonik markaları modern bir arayüzle sunan dinamik ve responsive (mobil uyumlu) bir web tasarımı projesidir.

## 🚀 Özellikler

* **Çok Sayfalı (Multi-page) Yapı:** Ana Sayfa, Tarihçe, Türler, Motorlar, Markalar, Galeri ve İletişim sayfalarından oluşan eksiksiz mimari.
* **Dinamik Segment İncelemesi:** Türler sayfasında JavaScript tabanlı akordeon (accordion) yapısı kullanılarak, motor tarzlarına tıklandığında alt modellerin teknik detayları ve görselleri pürüzsüz bir animasyonla açılır.
* **Responsive Tasarım:** CSS Grid ve Flexbox teknolojileri bir arada kullanılarak sitenin tüm mobil cihazlar, tabletler ve masaüstü ekranlarla %100 uyumlu çalışması sağlanmıştır.
* **İnteraktif İletişim Formu:** JavaScript ile güçlendirilmiş, form gönderildiğinde sayfa yenilenmeden dinamik başarı mesajı fırlatan kullanıcı dostu iletişim alanı.
* **Yerel Veri Yönetimi:** Proje, internet kısıtlamalarından etkilenmemesi ve maksimum hızda çalışması için tamamen yerel (local) görsel ve stil bağımlılıklarıyla kurgulanmıştır.

## 🛠️ Kullanılan Teknolojiler

* **HTML5:** Semantik (semantik) etiketler kullanılarak SEO ve tarayıcı dostu iskelet yapısı oluşturuldu.
* **CSS3:** Koyu tema (dark mode) odaklı, modern renk paletine sahip, geçiş efektli (transition) ve esnek yerleşimli modern tasarım kuralları uygulandı.
* **JavaScript (ES6):** Sayfa içi dinamik içerik gösterimi, DOM yönetimi ve form kontrolü için saf (Vanilla) JavaScript kullanıldı.
